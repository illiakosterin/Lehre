Graduate-level course on machine learning methods and algorithms for MSc Data Science students, with applied finance contexts throughout. Organised around regression, instance and ensemble methods, and advanced topics. Each lecture ships with slides, a Colab notebook, and a synthetic dataset — plus both technical and accessible versions of the material.

Topics

Category Topic Key Concepts
Regression Linear Regression Simple / multiple regression, gradient descent, regularisation, bias-variance tradeoff
Logistic Regression Sigmoid functions, decision boundaries, ROC curves, classification metrics
Instance & Ensemble KNN & K-Means Distance metrics, clustering, elbow method, silhouette analysis
Random Forests Decision trees, bootstrap aggregating, feature importance, ensemble voting
Advanced PCA & t-SNE Dimensionality reduction, scree plots, visualisation techniques
Embeddings & RL Word embeddings, Q-learning, reinforcement learning fundamentals
